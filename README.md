# Cine Lumière — Presentación Web

Presentación interactiva en React + Tailwind CSS + Framer Motion.

## Desarrollo local

```bash
npm install
npm run dev
```

## Cómo añadir la imagen de arquitectura

1. Genera o exporta el diagrama de arquitectura
2. Renómbrala como `arquitectura.png`
3. Cópiala a la carpeta: `public/images/arquitectura.png`
4. Ejecuta `npm run dev` — aparecerá automáticamente en el slide "Arquitectura General"

> Mientras la imagen no exista, se muestra un placeholder con instrucciones.

## Navegación

| Tecla | Acción |
|-------|--------|
| `→` / `↓` | Siguiente slide |
| `←` / `↑` | Slide anterior |
| Click en dots | Ir a slide concreto |
| Swipe | Navegar en móvil |
